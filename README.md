# Excel VBA KPI Automation Project

## Overview
This project automates daily KPI monitoring and reporting using Excel VBA. 
It was created to replace a manual reporting process that involved analyzing multiple Excel files and sending daily status updates.

## Problem
- Manual KPI checks were time-consuming
- High risk of human error
- No consistent visualization of trends

## Solution
The VBA automation:
- Consolidates KPI data from multiple Excel files
- Checks KPIs against predefined thresholds
- Highlights breaches using conditional formatting
- Generates 7-day trend charts for key KPIs
- Automatically creates and sends a formatted email report with charts and attachments

## Tools Used
- Microsoft Excel
- VBA (Visual Basic for Applications)

## Skills Demonstrated
- Data cleaning and transformation
- Automation and scripting
- KPI analysis
- Data visualization
- Reporting and stakeholder communication

## Note
This is a demo version using anonymized and sample data. 
The original project was implemented in a production environment.
